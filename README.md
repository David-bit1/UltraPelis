# UltraPelis

UltraPelis es una pagina web para ver peliculas online con catalogo centralizado en Supabase.

## Paso actual

Paso 4: migracion completa a Supabase.

## Reglas del proyecto

- Solo peliculas.
- No crear un archivo HTML por cada pelicula.
- Guardar la informacion del catalogo en Supabase.
- Mostrar los reproductores externos mediante `iframe`.
- Mantener un diseno oscuro, moderno y responsive.
